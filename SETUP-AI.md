# 接入 AI（百炼/通义千问）

> 这份 SOP 给试用 Aether 的同事：用阿里云百炼接入 AI，让 App 的对话、自动标签、文档问答能用起来。
> 全程约 5 分钟。

---

## 一、申请百炼 API Key

### 1. 注册并开通百炼

打开 **<https://bailian.console.aliyun.com/>** ，用阿里云账号登录（没有就先注册一个，支持手机号/支付宝）。

第一次进入会提示开通 **大模型服务**，点开通。**新用户有免费额度**，足够日常试用，不会立刻扣钱。

### 2. 创建 API Key

进入控制台后：

1. 左侧菜单找到 **API Key 管理**（或顶部头像 → API-KEY）
2. 点 **创建我的 API-KEY**
3. 给它起个名字，例如 `aether-mac`
4. 点确定 → **立刻复制 API Key**（`sk-` 开头那串）
5. 找个安全的地方存好（密码管理器、备忘录加密都行）

> ⚠️ **这个 Key 只显示一次**。关掉弹窗就再也看不到了，丢了只能重新建。

---

## 二、在 Aether 里配置

### 1. 打开设置

启动 Aether → 左下角找 **设置**（齿轮图标）→ 切到 **AI 模型** tab。

### 2. 添加模型

1. 点 **+ 添加 AI 模型**
2. 在表单顶部点 **「通义千问」** 这个胶囊按钮 —— 它会自动帮你填好 **Endpoint** 和 **模型名**
3. 手动填写剩下两个字段：

| 字段 | 填什么 |
|------|--------|
| 名称 | 随便起，例如 `千问 Plus` |
| 模型名 | 默认 `qwen-plus`，可改（见下方推荐） |
| Endpoint URL | 已自动填 `https://dashscope.aliyuncs.com/compatible-mode`，**不要动** |
| API Key | 粘贴你刚才从百炼复制的 `sk-xxx` |
| 最大 Token | 默认 2048 够用，长文档可调到 4096+ |

4. 点 **保存**

### 3. 测试连接

回到列表，找到刚加的模型，点旁边的 **测试** 按钮。

- ✅ 看到 **"连接成功"** → 配好了，可以用 AI 功能了
- ❌ 看到红色错误提示 → 跳到下面的「常见报错」排查

---

## 三、模型选什么

百炼有一堆模型，**改下表里的"模型名"字段**就能切换：

| 模型名 | 适合 | 价格（输入/百万 token） |
|--------|------|------------------------|
| `qwen-turbo` | 简单总结、标签生成 — 快又便宜 | ¥0.3 |
| `qwen-plus` | 日常对话、文档问答 — **推荐默认** | ¥0.8 |
| `qwen-max` | 复杂推理、深度分析 | ¥2.4 |
| `qwen-long` | 长文档（>32K token） | ¥0.5 |

价格仅供参考，最新看 <https://help.aliyun.com/zh/model-studio/billing-for-model-studio>。

---

## 四、安全提示

- API Key 在 Aether 里通过 macOS Keychain 加密存储（用的是 Electron `safeStorage`），不会明文落盘
- 不要把 Key 截图发到群里、贴到聊天工具
- 如果不小心泄露，去百炼控制台立刻吊销那个 Key，重建一个

---

## 五、常见报错

### `401 Unauthorized` / `Invalid API key`
API Key 输错了。回设置面板 → 删除这个模型 → 重新添加，确认复制时没多/少字符。

### `429 Too Many Requests`
触发限流。新账号默认 QPM (每分钟请求次数) 有限，等几秒重试。频繁触发就去百炼控制台申请提升限额。

### `404 Not Found` / `model not exist`
模型名拼错了。检查"模型名"字段是不是 `qwen-plus` / `qwen-max` 之类的，不要加引号或空格。

### `Network error` / 连接超时
- 检查网络（国内访问百炼不需要 VPN，反而开了 VPN 可能更慢）
- Endpoint URL 是不是被改成奇怪的值了（应该是 `https://dashscope.aliyuncs.com/compatible-mode`）

## 六、配好后能用什么

- **文档问答**：打开一个 .md 文件 → 右侧面板 → AI 对话 tab → 提问
- **批量功能**（后续版本）：自动生成对话记录的标题和标签、生成周报

如果你发现某个 AI 功能 App 里没有，那是因为还在开发中，给反馈给开发者（@TongWu021）。

## 附：用其他模型？

DeepSeek、Moonshot、本地 Ollama 都支持，添加模型时点对应的预设按钮即可。API Key 申请方式各家不一样，自己 Google。
